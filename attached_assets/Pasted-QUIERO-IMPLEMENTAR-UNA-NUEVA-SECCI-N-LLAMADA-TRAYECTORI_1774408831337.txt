7. ORDEN EXACTO DE RENDERIZADO DEL PERFIL
==================================================

Cuando se muestre un candidato, renderizar en este orden EXACTO:

--------------------------------------------------
A) HEADER DEL CANDIDATO
--------------------------------------------------

Mostrar:
- nombre completo
- partido
- edad
- profesión
- resumen
- formación académica

La formación académica debe mostrarse como lista clara y limpia.

--------------------------------------------------
B) HISTORIAL JUDICIAL
--------------------------------------------------

Esta sección debe mostrarse PRIMERO entre los bloques de historial.

Debe tener un fondo apenas diferenciado, muy sutil, serio y profesional, no alarmista.

Para cada caso mostrar:
- título del caso
- tipo (badge neutro)
- estado con badge de color
- descripción completa
- fuentes en texto pequeño

Mapa de colores de estado:
- `archivado` → verde
- `absuelto` → verde
- `activo` → amarillo o naranja
- `condena` → rojo

Usar `estado_label` si existe.
Si no existe, usar `estado`.

Si no hay registros judiciales:
mostrar exactamente:
`Sin registros judiciales documentados`

--------------------------------------------------
C) HISTORIAL POLÍTICO
--------------------------------------------------

Mostrar los items en orden cronológico inverso:
- más reciente primero

Cada item debe mostrar:
- cargo
- partido
- periodo
- descripción

--------------------------------------------------
D) HISTORIAL PROFESIONAL
--------------------------------------------------

Mostrar los items en orden cronológico inverso:
- más reciente primero

Cada item debe mostrar:
- cargo
- institución
- periodo
- descripción

--------------------------------------------------
E) CONTROVERSIAS / CONTEXTO SOCIAL
--------------------------------------------------

Cada item debe mostrar:
- título en negrita
- descripción objetiva

--------------------------------------------------
F) PARTIDO POLÍTICO
--------------------------------------------------

Mostrar:
- nombre
- símbolo
- ideología
- fundación
- historia
- desempeño electoral

--------------------------------------------------
G) DATOS IMPORTANTES
--------------------------------------------------

Mostrar como lista de puntos destacados.

--------------------------------------------------
H) LÍNEA DE TIEMPO
--------------------------------------------------

Renderizar los hitos del candidato ordenados cronológicamente.

Puede ser vertical u horizontal, pero debe:
- verse limpia
- ser clara
- verse bien en desktop y mobile
- mantener coherencia total con el resto del sitio
