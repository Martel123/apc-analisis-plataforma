DIAGNÓSTICO
=====================================

=====================================
REGLAS OBLIGATORIAS
=====================================

1. UN BLOQUE SOLO SE ABRE UNA VEZ
Si ya existe un bloque con el mismo nombre normalizado, no abrir otro.

2. UNA VARIABLE SOLO SE ABRE UNA VEZ
Si ya existe una variable con el mismo número o nombre normalizado, no duplicarla.

3. NORMALIZAR NOMBRES ANTES DE COMPARAR
Debes normalizar:
- mayúsculas/minúsculas
- espacios dobles
- tildes si hace falta
- guiones largos
- y ELIMINAR puntajes pegados al nombre

Ejemplo:
- "ECONOMÍA"
- "ECONOMÍA — 6.0/10"
- "BLOQUE 1: ECONOMÍA — Puntaje promedio: 6.0/10"

Todo eso debe mapear al mismo bloque base:
ECONOMÍA

Igual para variables:
- "Política tributaria"
- "Variable 1: Política tributaria"
- "Política tributaria — 8.3/10"

Todo debe mapear a la misma variable base.

=====================================
FILTROS OBLIGATORIOS
=====================================

NO debes tratar como bloque o variable:
- filas de tabla
- líneas resumen
- recapitulaciones
- títulos acompañados solo por puntaje
- secciones de promedio del bloque
- listados tipo cuadro D/P/M/Im/V/E
- repeticiones de nombres en cuadros de cierre
